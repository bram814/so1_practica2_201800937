# Backend en Nodejs


CREAR package.json

```bash
npm init -y
```

Dependencias

```bash
npm install express morgan oracledb cors
````

Por último instalar la libería nodemon para que cada vez que se actualice en tiempo real.

```bash
npm i nodemon
````


## Install Mysql en Nodejs

```bash
npm i mysql
```


install dotenv, para archivos .env

```bash
npm i dotenv
````
