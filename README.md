##### How to install Nodejs on Ubuntu
____


Install Nodejs and npm
```bash
sudo apt install nodejs npm
```

Version Nodejs and npm

```bash
sudo node -v && npm -v
# v18.7.0
# 8.17.0
```


Create React App
```bash
npx create-react-app my-app
cd my-app
npm start
```
